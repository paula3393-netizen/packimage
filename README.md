# pack-image

Description. 
The package pack-image is used to:
	Processing:
		- Histrogram matching
		- Structkural similarity
		- Resize image 
	Utils:
		- Read image
		- Save image
		- Plot image
		- Plot histogram

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install package_name

```bash
pip install packimage
```
